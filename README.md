Contents
--------------------------------

     This repository contains supplementary R code for the article:  

     'Principled type I error rate inflation in two-arm clinical trial designs with external control information borrowing'. 

     by Silvia Calderazzo, Manuel Wiesenfarth, Vivienn Weru and Annette Kopp-Schneider.  


Notes
--------------------------------

The code is meant only to allow reproduction of the designs and figures of the above article. Note that computation of the designs is by default parallelised via the ``mclapply`` function of the ``parallel`` R package.
